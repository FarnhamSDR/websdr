SUWS WebSDR

Online WebSDR for 23cm, and soon more!

This WebSDR is operated by the Southampton University Wireless Society in collaboration with Martin G8JNJ and Noel G8GTZ.

Information on the setup and equipment can be found on the WebSDR Wiki Page.

The software used for this WebSDR is kindly provided by the WebSDR Project.
Note: you need both Java and JavaScript enabled for this page to work properly. If you don't hear anything, probably Java is disabled or its version is too old (i.e., pre-1.4.2).
